# /morning-brief

Perform a start-of-day context restore for this project.

## Steps

1. **Read context files** — Read `CLAUDE.md` (project root), `PROGRESS.md`, `frontend/MIGRATION.md`, and `server/MIGRATION.md`.

2. **Project summary** — In 3–5 sentences, describe what SlimeList is and the current state of the refactor/feature work.

3. **Where we left off** — Summarize the last completed task and any partial work from the most recent `PROGRESS.md` entry.

4. **Immediate next task** — State the single next action to resume work. Be specific: which file, which phase, which component.

5. **Unresolved flags** — List anything marked as unresolved, blocked, or deferred in `PROGRESS.md` or the MIGRATION files.

Do not modify any files. This command is read-only.
